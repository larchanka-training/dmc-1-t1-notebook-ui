# Modern Software Development Student Template

Simple React + TypeScript template for course projects.

## Quick start

```bash
npm install
npm run dev
```

Open the local URL from Vite (usually `http://localhost:5173`).

## Scripts

- `npm run dev` - start development server
- `npm run build` - type-check and create production build
- `npm run preview` - preview production build locally
- `npm run lint` - run ESLint checks

## Suggested extension points

- Add feature folders in `src/` (for example `src/features/todos`).
- Create reusable UI components in `src/components`.
- Add API helpers in `src/services`.
- Introduce tests (`vitest`, `react-testing-library`) when your project scope is clear.
